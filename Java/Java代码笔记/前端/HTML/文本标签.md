#块状标签和内联标签
常见块状标签：段落<p>, 标题<h1>,<div>
常见内联标签：<span>, 图片<img>, <strong>
```
<!-- h1-h6是块状标签 -->
<h1>Beautiful World</h1>

<!-- h2是h1的子标题 -->
<h2>Islands</h2>

<!-- h3是h2的子标题 -->
<h3>Maldives</h3>
<h3>Crete Island</h3>

<h2>Mountains</h2>
<h3>Matterhorn, Swiss-Italian Border</h3>
<h3>Kirkjufell</h3>
```
